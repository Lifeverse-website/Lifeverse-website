# LifeVerse Technologies Website

Professional starter website for LifeVerse Technologies with:

- Public company pages
- Public project portals for LifeVerse RP, LifeCoin, and Lifetime
- Document download library
- Free starter admin portal with local backup export/import
- JSON templates for records and backups

## Upload instructions

1. Replace the files in your GitHub repository with the contents of this package.
2. Keep the folder names exactly as they are.
3. Make sure GitHub Pages is still set to deploy from `main` and `/ (root)`.
4. Open the live website and test each page.
5. Use the admin portal for simple local record keeping until you upgrade to a database later.

## Important note

The admin portal in this free version stores data in the browser using local storage. Use the export backup button regularly.
