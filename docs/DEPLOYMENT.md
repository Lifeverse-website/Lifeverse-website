# LifeVerse Technologies Deployment Guide

## GitHub structure
Upload the full project into the repository root with this structure:

- index.html
- css/styles.css
- js/app.js
- assets/img/
- docs/

## Cloudflare Pages settings
Use these exact settings in Cloudflare Pages:

- Framework preset: None
- Build command: leave empty
- Build output directory: .

## Domain
Once deployment is successful, connect your custom domain from the Cloudflare Pages dashboard.

## Contact form
The contact form uses FormSubmit and sends to:

lifeverse_technologies@zohomail.com

### Important first activation step
After deploying:
1. Open the live site
2. Submit the contact form once
3. Check the Zoho inbox
4. Confirm the FormSubmit activation email

Without that first confirmation, normal submissions will not continue.
